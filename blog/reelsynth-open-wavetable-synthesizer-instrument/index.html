ReelSynth: An Open Wavetable Synthesizer Instrument for Inspectable Sound Design

Most production stacks treat synthesis as a black box: you pick a preset, tweak two macros, and hope the render survives the mix. ReelSynth is different — it is a wavetable synthesizer instrument with inspectable DSP, open formats, and a voice engine you can route, modulate, and export without losing authorship.

Instrument, not plugin theater

ReelSynth is classified as an instrument in the Reeldemo stack: a sound source you sculpt directly, not a utility that hides its math. Tri-osc wavetable voices, ADSR envelopes, state-variable filters, an eight-slot modulation matrix, and a reorderable FX rack are all addressable parameters — not marketing bullet points behind a locked UI.

The engine ships as MIT Rust with a Python wheel for offline rendering inside Reeldemo Studio. Standalone egui UI and a plugin shell are on the roadmap; the DSP and patch schema are production-ready today.

Wavetables you can actually own

Import Vital .vitaltable banks, WAV single-cycle folders, and Serum .fxp wavetable subsets. Export to Vital, WAV frames, Serum RSWT, Ableton param JSON, SFZ, MIDI, 24-bit audio, or a reelpack/ bundle. Patches live in .reelpreset JSON; banks in .reelwt binary — both documented and versioned.

Where it sits in the stack

  • Reeldemo Studio — agentic compose and grade; ReelSynth renders offline stems and patches.
  • ReelSynth — hands-on wavetable synthesizer instrument for sound design.
  • Kaleidoscope — ASCII promos that match the dark studio aesthetic.

If you need speed and editability under pressure, start with an instrument whose parameters survive the handoff — not a render-and-pray preset chain.
