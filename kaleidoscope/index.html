Kaleidoscope ASCII

Fast kaleidoscope mirror and ASCII art engine for Reeldemo promos, social ads, and landing visuals. Sub-millisecond renders at terminal scale — the same engine powers the hero on this site.

Parallel pixel sampling in Rust, exposed through npm native bindings and a CLI. Turn stills into hypnotic symmetry without leaving the terminal.

Parallel sampling with configurable segments, rotation, and palette mapping.

npm package
